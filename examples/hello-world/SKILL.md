# Hello World Skill

A simple example skill to demonstrate basic structure and functionality.

## Description

This is a minimal skill that demonstrates the basic structure and components required for a Claude Code skill. It serves as a learning tool for understanding how skills work.

## Capabilities

- Respond to hello world requests
- Demonstrate basic skill structure
- Show example formatting

## When to Use

Use this skill when:
- Learning how Claude Code skills work
- Testing skill installation
- Creating a template for new skills

## Instructions

### Main Workflow

1. **Detect Request**
   - Listen for "hello world" or similar greetings
   - Recognize test or example requests

2. **Respond**
   - Provide a friendly greeting
   - Explain what this skill does
   - Offer to help with skill creation

3. **Optional Actions**
   - Show skill structure
   - Demonstrate file operations
   - Explain best practices

## Examples

### Example 1: Basic Hello

**User Request:**
"Run the hello world skill"

**Expected Behavior:**
- Detect the hello world request
- Respond with a friendly message
- Explain the skill's purpose

### Example 2: Learning Request

**User Request:**
"Show me how this skill works"

**Expected Behavior:**
- Explain the skill structure
- Point to key components (SKILL.md, examples)
- Suggest next steps for learning

## Notes

- This is a minimal example for learning purposes
- Real skills should have more substantial functionality
